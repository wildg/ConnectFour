# Web Connect Four

Created this game as a final project for my web-based systems class. The game connects the user using Socket.IO, and uses Node.JS for the backend. The following steps are used to begin the program:

1. Ensure that you have the latest version of `npm`
2. Get Socket.IO by running `npm install socket.io` in the project folder
3. Run `node index.js`
4. Go to `http://localhost:3000` on your machine
5. Have fun!
